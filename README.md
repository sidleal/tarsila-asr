# Tarsila-ASR: A Multi-Domain Test Suite for Benchmarking Brazilian Portuguese Speech Recognition

| Folder | Content |
| :--- | :--- |

| whisper-large | Scripts for fine-tuning Whisper Large and Medium.  |
